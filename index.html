___________________________


HTML (HyperText Markup Language) é a base de todos os sites. Utilizo HTML para criar a estrutura de páginas web, garantindo que o conteúdo seja semântico e acessível, consigo organizar e estruturar de maneira clara e eficiente.


CSS (Cascading Style Sheets) me permite estilizar sites, criando layouts responsivos e designs atrativos. , consigo oferecer uma experiência visual agradável em diferentes dispositivos. Adicionalmente, utilizo as transições para criar efeitos dinâmicos e interativos.


JavaScript é uma linguagem de programação que permite adicionar interatividade a sites. Ele é executado no navegador do usuário e permite modificar o conteúdo da página, responder a ações do usuário e se comunicar com servidores para carregar dados dinâmicos.


jQuery é uma biblioteca do JavaScript que facilita a manipulação do DOM e o controle de eventos. Usando jQuery, consigo criar animações suaves e interações mais complexas com menos código, além de garantir compatibilidade entre diferentes navegadores.
